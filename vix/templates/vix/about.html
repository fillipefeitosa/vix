About Vix

VIX is small (and minimal too!) client for Cuckoo sandbox that uses Malheur to complement the sandbox analisys with Malware behavior.

With VIX you can submit files, ex.: an e-mail atachment attachment, and get a feedback to ensure if it is a safe file.


How VIX works?

As a client, VIX organize things given by Cuckoo and Malheur. Using MIST as pattern to malware behavior input, there is a Machine Learning process to determine if a binary has a similar behavior compared with a know malware from database. Each time a malware is submited, it adds knowledge to the database. And, well, knowledge is power!

There are two expected returns if a file is somehow malicious:

  • Signature: VIX give you the MalScore(from Virustotal) and the percentage of antivirus softwares that identified the file as malware.
  • Behavior: VIX give you the closest malware and the group (cluster) that behave like the submmited file.
